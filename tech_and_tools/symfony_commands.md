# <p align="center">**_Symfony Commands_**</p>
<br>

## **1. Symfony Server**

1. Server run    

        symfony server:start
2. Server run in backround

        symfony server:start -d
3. Server stop

        symfony server:stop
<br>

## **2. Cache Clear**

    php bin/console cache:clear
<br>

## **3. User Password Change**

    php bin/console fos:user:change
<br>
