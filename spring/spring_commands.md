# **Spring Commands**

1. **Build an executable JAR file**

    > For Maven

        mvn clean install

    > For Gradle

        gradle clean build

2. **Run JAR file**

        java -jar [jar_file]

3. **Run Maven project in command prompt**

        i) Linux/MacOS: ./mvnw spring-boot:run
        ii) Windows: mvnw spring-boot:run
